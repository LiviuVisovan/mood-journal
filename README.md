# Mood Journal

A small React + TypeScript app 
Select a mood then write a note and save it.  
Each entry are stored in localStorage and can be edited, deleted, filtered, and sorted.

## Features
- Choose a mood from a dropdown
- Write and save a note
- Edit and delete entries
- Search filter
- Sort by newest/oldest/mood
- Data saved in localStorage

## Tech
- React
- TypeScript
- Vite

## Run locally
npm install
npm run dev
